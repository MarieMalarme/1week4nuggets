# 1 week = 4 nuggets

An inspirational depository showcasing 4 nuggets about design or code per week:

- 1 event: talk, conference, exhibition...
- 1 project: identity, sketch, website...
- 1 book: book, fanzine, magazine...
- 1 quote: phrase, tweet, word...
